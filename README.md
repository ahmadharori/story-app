## List of Features
1. Welcome Page
2. Signup
3. Login
4. Story List
5. Language Settings (for localization)
6. Story Map
7. Add Story (with CameraX and Intent Gallery)

7. Kotlin Coroutines
8. RecylcerView, CardView
9. Datastore, preferences
